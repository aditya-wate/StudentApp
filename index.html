About the project

The University - Department of Computer Science.
This project contains a Local Database with Students and Courses tables. The Data from these tables is fetched by WebForms .aspx pages here. The requirement was to display the Student records alongwith the Average grades which should be calculated dynamically. Additionally, Courses table records have been displayed on another page just to add more functionality and to go with the Design of a standard "University" website.
